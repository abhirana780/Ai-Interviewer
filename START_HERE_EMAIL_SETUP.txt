                        üöÄ QUICK START (5 STEPS)
================================================================================

STEP 1: Get Gmail App Password
-------------------------------
1. Visit: https://myaccount.google.com/apppasswords
2. Sign in to Gmail
3. Enable 2-Step Verification (if asked)
4. Select "Mail" ‚Üí "Windows Computer" ‚Üí "Generate"
5. Copy the 16-character password (example: abcd efgh ijkl mnop)
6. Remove spaces: abcdefghijklmnop


STEP 2: Edit .env File
----------------------
1. Open: d:\wipronix\.env
2. Find this line:
   EMAIL_ENABLED=false
   
   Change to:
   EMAIL_ENABLED=true

3. Find this line:
   SMTP_USERNAME=
   
   Change to:
   SMTP_USERNAME=your-email@gmail.com
   (Replace with YOUR Gmail address)

4. Find this line:
   SMTP_PASSWORD=
   
   Change to:
   SMTP_PASSWORD=abcdefghijklmnop
   (Paste the 16-character code from Step 1, NO SPACES!)

5. Find this line:
   SMTP_FROM_EMAIL=
   
   Change to:
   SMTP_FROM_EMAIL=your-email@gmail.com
   (Same as SMTP_USERNAME)

6. SAVE the file (Ctrl+S)


STEP 3: Restart Server
----------------------
1. Press Ctrl+C in the terminal to stop server
2. Run: python run.py
3. Look for: ‚úÖ Loaded email configuration from .env file
   (If you see this, configuration is loaded successfully!)


STEP 4: Test Email
------------------
1. Go to: http://127.0.0.1:7860/admin
2. Click any candidate's "View" button
3. Click "üìß Send Results Email"
4. Click "OK" to confirm
5. Wait 10 seconds


STEP 5: Verify Email Received
-----------------------------
1. Check candidate's email inbox
2. Look for subject: "Your Interview Results - [Track Name]"
3. If not in inbox, check spam/junk folder
4. Email should arrive within 1-2 minutes

================================================================================
                        ‚úÖ EXAMPLE .env FILE
================================================================================

After editing, your .env file should look like this:

EMAIL_ENABLED=true
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587

REPLACE:
- john.doe@gmail.com ‚Üí YOUR Gmail address
- abcdefghijklmnop ‚Üí YOUR 16-character App Password
