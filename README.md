# AI Interviewer\n\nA local Flask-based AI interviewer that asks technical questions, evaluates answers, and provides follow-ups. Includes a simple frontend with camera/mic recording, optional speech-to-text, and TTS for the interviewer. No paid APIs required.\n\n## Features\n- Interview tracks (MERN, Data Science, Analytics, AI/ML, Python, Java, Cloud, Cyber)\n- Video/audio answer capture and upload (WebM)\n- Basic scoring and feedback\n- Transcript persistence in SQLite\n- Optional TTS for questions\n\n## Quick Start\n\n### Windows\n1) Ensure Python 3.10+ installed.\n2) In PowerShell, from the project root:\n   - Option A: run `run.bat` (creates venv, installs deps, starts server)\n   - Option B: manual:\n     - `python -m venv venv`\n     - `venv\\Scripts\\activate`\n     - `pip install -r backend\\requirements.txt`\n     - `python backend\\app.py`\n\n### macOS/Linux\n1) Ensure Python 3.10+ installed.\n2) From the project root:\n   - `chmod +x run.sh`\n   - `./run.sh`\n\nServer runs at: http://127.0.0.1:7860/\nOpen that URL to use the frontend. Choose a track and click Start, then Record/Submit answers.\n\n## API Endpoints\n- POST /start\n  - Body: `{ "track": "MERN" }` or `{ "role": "Software Engineer" }`\n  - Returns: `session_id`, `question`, `bot_video_url` (optional), `bot_image_url`, `tts_url` (optional)\n- POST /answer\n  - Form-data: `session_id`, optional `media` (webm), `answer` (text)\n  - Returns: `next_question`, `score`, `feedback`, plus media and bot asset URLs\n- POST /tts\n  - Body: `{ "text": "Hello", "session_id": "..." }`\n  - Returns: `tts_url`\n- GET /transcript/<session_id>\n- GET /health\n\n## Models\nPlace local models in `model/models/`. If using GPT4All, put a `.gguf` file (e.g. `gpt4all-model.gguf`) there. Model files are ignored by Git by default; use Git LFS if you prefer tracking them.\n\n## Data & Media\n- Database: `backend/database/interviews.db` (auto-created, gitignored)\n- Uploaded answers: `static/media/answers/<session_id>/...` (gitignored)\n- TTS audio: `static/media/tts/<session_id>/...` (gitignored)\n- Bot avatar: `static/media/bot.svg` (tracked)\n\n## Project Structure\n- `backend/` Flask app, interviewer logic, DB helpers\n- `frontend/` minimal HTML/JS page (served at `/`)\n- `static/` CSS/JS and generated media\n- `model/` local model files\n\n## Notes\n- Default port is 7860. Adjust in `backend/app.py` if needed.\n- If camera/mic access fails, the app allows text-only answers.\n- For production behind a proxy (Nginx/Caddy), expose only the Flask server and static assets.\n
